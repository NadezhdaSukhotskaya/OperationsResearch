ЛАБОРАТОРНАЯ РАБОТА №1 реальзовывана на языке python. ОС windows
 Чтобы выполнить программу, необходимо уставновить библиотеки с помощью слудующей команды:
 pip install -r requirements.txt
 Чтобы запустить необходимо выполнить слудующую команду:
 python main.py.
